## 👋 About Me

I'm a developer based in ShenZhen, building on the Android ecosystem since 2016. My work centers on Kotlin, Jetpack Compose, and clean architecture patterns. I focus on practical tooling — from lightweight network abstractions to native HTTP handling in C — and maintain several open-source projects that reflect a builder's mindset: high commit activity, minimal overhead, and a preference for shipping working code over extensive issue tracking.

- **Active in**: Kotlin, Android, Jetpack Compose, MVVM, architecture components
- **Recent focus**: notes-app, compose, coroutines, CameraX, Flutter exploration
- **Dominant mode**: Builder — 100% commits, 0% PR/issue/review noise
- **Member since**: 2016
